# Windows Sub-System for Linux (WSL)

From Windows:

```powershell
wsl --install -d Ubuntu
wsl --set-default-version 2
wsl --status
wsl --update
wsl --export
wsl --import
wsl --exec <command>
```

From Linux:

```bash
sudo apt update && sudo apt upgrade && sudo apt autoclean

sudo apt install build-essential

sudo apt update && sudo apt upgrade && sudo apt autoclean
```
