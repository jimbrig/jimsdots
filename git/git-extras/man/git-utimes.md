git-utimes(1) -- Change files modification time to their last commit date
=========================================================================

## SYNOPSIS

`git-utimes`

## DESCRIPTION

  Change files modification time to their last commit date.

## OPTIONS

  No options needed.

## EXAMPLES

  git utimes
